# Tutorial de Uso do OtoManus

Este tutorial irá guiá-lo através dos passos básicos para usar o OtoManus.

## 1. Iniciando uma Conversa

- Na tela principal, digite sua solicitação na caixa de texto e pressione Enter.
- O agente irá analisar sua solicitação e começar a trabalhar.

## 2. Gerenciando Arquivos

- Clique na aba "Arquivos" para ver os arquivos no workspace do agente.
- Clique em um arquivo para ver seu conteúdo.

## 3. Configurando o Agente

- Clique na aba "Configurações" para alterar as configurações do LLM, navegador e busca.
- Clique em "Salvar Configurações" para aplicar as mudanças.

## 4. Adicionando Ferramentas e MCPs

- Na aba "Ferramentas", você pode ativar ou desativar as ferramentas disponíveis.
- Na aba "MCP Servers", você pode adicionar novos servidores MCP para estender as capacidades do agente.

## Exemplo de Uso

**Solicitação:** "Crie um arquivo chamado `hello.txt` com o conteúdo `Hello, Otomanus!`"

O agente irá:
1.  Selecionar a ferramenta `str_replace_editor`.
2.  Criar o arquivo `hello.txt` com o conteúdo especificado.
3.  Você poderá ver o arquivo na aba "Arquivos".
